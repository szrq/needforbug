飞飞表情插件v1.0_[普通表情/魔法表情]

文章地址：http://www.ffasp.com/content.asp?newsid=1722  关注Web前端开发 - ffasp.com

说明：
本插件是以新浪微博的表情发布功能为基础，内含 两大类表情：普通表情和魔法表情，而且两类表情都含有简体/繁体两个版本，本插件完美支持IE6,IE7,IE8,firefox3,firefox4浏览器。

其中：

普通表情包含：默认/心情/爱情/休闲/搞怪/管不着/大耳兔/哈皮兔/天气/星座/小新小浪/亚运会/张小盒/悠嘻猴/阿狸

魔法表情包含：默认/心情/搞怪/浪漫爱情/时尚/闯堂兔/小新小浪/张小盒/旺狗/悠嘻猴/阿狸

以上表情均含有简繁体双语，表情显示使用javascript的客户端浏览器替换。

文件目录说明：

css 样式目录

　　－bgimg 背景图片目录

　　－emotion.css 该插件使用的样式文件

image 图片目录

　　－emotions 表情文件目录

　　　　－magic 魔法表情目录

　　　　－normal 普通表情目录

js 文件目录

　　－emotion.js 表情插件文件

　　－emotion_data.js 表情数据文件(数组格式)

　　－jquery.min.js jquery类库

　　－swfobject.js swf播放插件

　　－zh_cn.js 简体中文语言包

　　－zh_tw.js 繁体中文语言包

 

使用方法：

1.语言版本的切换：js目录内有2个语言文件：zh_cn(简体)和zh_tw(繁体)，页面中加载对应的语言文件即可切换语言

2.选择表情调用方法：

$('#test').showEmotion({input:$('#publish_text2')});

tese为点击按钮的id,input为表单输入框：textarea或者input

普通表情：

[愛你][萌][发芽][强吻]

魔法表情：

[愛你][萌][发芽][强吻]

 

3.显示表情调用方法

$('#fftest').listEmotion();

fftest为显示表情div的id;
