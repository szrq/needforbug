关于浏览器

DYHB.BLOG_X 2.0.1有部分低版本浏览器无能友好地支持。目前我们发现IE6（包括以IE6为内核的360和搜狗浏览器）下编辑器无法加载，IE8下Ajax丢失COOKIE。我们建议，进入博客后台请使用Google Chorme，Firefox以及IE7,9,10等浏览器版本，关于浏览器引起的功能问题一概不予解答。DYHB.BLOG_X 2.0.1目前官方前台模板均兼容IE6等主流浏览器。
